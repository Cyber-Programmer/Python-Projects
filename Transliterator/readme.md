# Transliterator
This Python script allows users to transliterate text between English and Urdu. It provides an interactive menu for users to choose the direction of transliteration or to end the program.

# Features
**Interactive Menu:**

Users can choose to transliterate from English to Urdu, Urdu to English, or end the program.

**Transliteration:**

Converts English text to Urdu script using a predefined mapping.

Converts Urdu text to English script using a predefined mapping.

**Continuous Operation:**

The script runs in a loop, allowing multiple transliterations until the user chooses to end the program.
